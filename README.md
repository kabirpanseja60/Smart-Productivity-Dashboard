# 🚀 SmartDash – Productivity Dashboard

> Stay organized. Stay productive. Stay ahead.

SmartDash is a modern and responsive productivity dashboard designed to help users efficiently manage their tasks. Built using **HTML, Bootstrap 5, and JavaScript**, it provides a clean UI with powerful features like task tracking, filtering, statistics, and dark mode.

---

## 📌 Features

* ✅ Add, delete, and manage tasks
* 📊 Real-time task statistics (Total, Pending, Completed)
* 🔍 Filter tasks (All / Pending / Completed)
* 🎯 Priority levels (Low, Medium, High)
* 🌙 Dark mode toggle
* 🔔 Interactive alert notifications
* 📋 Task history table view
* 🎨 Clean and responsive UI using Bootstrap
* 💡 Motivational quotes carousel

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3
* **Framework:** Bootstrap 5
* **Icons:** Bootstrap Icons
* **Logic:** Vanilla JavaScript

## ⚙️ How to Run

1. Download or clone the repository:

   ```bash
   git clone https://github.com/your-username/smartdash.git
   ```

2. Open the project folder

3. Run the app:

   * Simply open `index.html` in your browser

---

## 🎯 Usage

* Click **"Add New Task"** to create tasks
* Use filters to view **Pending / Completed tasks**
* Mark tasks as complete or delete them
* Toggle **Dark Mode** for better UI experience
